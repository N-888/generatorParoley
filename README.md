## 🎯 О проекте

**Generator Paroley** — это современный CLI-менеджер паролей, созданный для безопасного хранения и генерации паролей. Приложение работает в командной строке и предоставляет полный контроль над вашими ключами доступа.

### Ключевые преимущества

- 🔒 **Безопасное хранение** — пароли хранятся в зашифрованном формате
- 🎲 **Генератор паролей** — создание надёжных паролей любой сложности
- 🚫 **Исключение похожих символов** — убирает 0/O, l/I для читаемости
- 📱 **Кроссплатформенность** — работает на Windows, macOS, Linux
- 📝 **Подробные комментарии** — код понятен даже новичкам

## 🚀 Быстрый старт

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/N-888/generatorParoley.git

# Перейти в папку проекта
cd generatorParoley
```

### Запуск

```bash

- Стандартная библиотека Python (дополнительные библиотеки не нужны)

### Требования к паролям

```
  - Минимум 8 символов
  - Хотя бы 1 буква (a-z, A-Z)
  - Хотя бы 1 цифра (0-9)
  - Хотя бы 1 спецсимвол: ! @ # $ % ^ & * ( ) _ + - = [ ] { } | ; : , . < > ?
  - НЕЛЬЗЯ: 2+ одинаковых подряд (aa, 11, !!)
  - НЕЛЬЗЯ: 2+ похожих из одной группы (0+O, l+I+1+!)
```

### Примеры проверки пароля

```

## 🔧 Конфигурация

### Настройки генерации

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `length` | 16 | Длина пароля |
| `exclude_similar` | True | Исключить похожие символы |
| `use_special` | True | Использовать спецсимволы |

### Исключённые символы

| Символ | Похож на | Причина |
|--------|----------|---------|
| `0` (ноль) | `O` (буква О) | Легко перепутать |
| `l` (строчная Л) | `I` (заглавная I) | Легко перепутать |
| `1` (единица) | `l` (строчная Л) | Легко перепутать |

## 🛡️ Безопасность

### Рекомендации

1. **Длина пароля** — минимум 12 символов
2. **Сложность** — комбинация букв, цифр, спецсимволов
3. **Уникальность** — разные пароли для разных сервисов
4. **Регулярная замена** — обновляйте пароли каждые 3-6 месяцев

### Оценка надёжности

Приложение автоматически проверяет пароль:

| Уровень | Описание |
|---------|----------|
| 🔴 Слабый | Длина < 8, нет спецсимволов |
| 🟡 Средний | Длина 8-11, есть буквы и цифры |
| 🟢 Хороший | Длина 12+, есть спецсимволы |
| 💚 Отличный | Длина 16+, все типы символов |
